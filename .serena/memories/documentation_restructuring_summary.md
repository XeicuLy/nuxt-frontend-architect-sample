## アーキテクチャ設計の強調

### ハイブリッド状態管理の明確化

```typescript
// 統合された使いやすいインターフェース
const {
  isLoading, // TanStack Query から
  healthStatusData, // TanStack Query から
  sampleInput, // Pinia から
} = useHealth();
```

### 責任の分離

- **TanStack Query**: サーバー状態（API データ、キャッシング、同期）
- **Pinia**: クライアント状態（UI状態、ユーザー入力、ローカルデータ）
- **統合レイヤー**: 両方を統合した使いやすいインターフェース
